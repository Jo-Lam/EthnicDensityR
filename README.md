This is the R script I used to prepare the manuscript: What does ethnic density represent? Spatial co-occurrence networks of a widely used contextual measure using harmonised UK small-area census data
The UK unified census data product is available to download from the Geographic Data Service at https://data.geods.ac.uk/dataset/unified-uk-census-data. The code pipeline to produce the product is available at https://github.com/GeographicDataService/unified-uk-census-2021-22/.
